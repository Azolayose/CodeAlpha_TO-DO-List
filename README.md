# My To-Do List Web App 

## Project Description
This is a simple To-Do List web application built using HTML, CSS, and basic JavaScript.  
It allows users to add, edit, delete, and mark complete tasks. The tasks are saved using local storage.

---

##  Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Data saved in browser (localStorage)
- Responsive design

---

##  Technologies Used
- HTML
- CSS
- JavaScript

---

##  How it works
1. Type a task in the input box
2. Click "Add"
3. Click task to mark as done
4. Use Edit/Delete buttons to manage tasks
